## 📃Descrição
### Programa para organizar tarefas e projetos.
### Organize tarefas por projeto, defina um prazo para cada tarefa e torne seu dia mais produtivo com o ToDo-APP.
<br>

## 🔧 Funcionalidades
- Criação projeto
- Visualizar projeto
- Edição projeto
- Exclusão projeto
- Criação tarefa
- Marcar tarefa como concluída
- Filtros de tarefas
- Edição tarefa
- Visualizar tarefa
- Ligar alarmes
- System tray
<br>



## ✔️ Técnicas e tecnologias utilizadas
- Java 17
- Maven 3.8.6
- Swing-UI
- SQLite
- jdatepicker
